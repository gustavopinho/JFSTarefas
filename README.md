# JSFTarefas
Implementação de um programa de lista de tarefas em Java.

## Funcionalidas

- Cadastro de usuário
- Login
- Cadastro de tarefa
- Exclusão de tarefa
- Alteração de tarefa
- Listagem de tarefas pendentes
- Listagem de tarefas finalizadas

## Dependências

- JSP 2.2
- JPA 2.1
- Postgresql
